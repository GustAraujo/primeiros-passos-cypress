# Minha automação com Cypress

Esse projeto é minha primeira experiência com automação em cypress, nele fiz testes end-2-end em um sistema de login e de usuário, todo o projeto foi feito baseado em JavaScript, utilizando o site OrangeHRM em tempo real.

## instalação 

```bash
npm install
```

>**NOTE**
>
> É necessário ter o **Node** instalado

## Rodando

```bash
# abrindo a interface do cypress

npx cypress open

## Rodando a automação na linha de comando

npx cypress run
